## Use Case 001 - Play Game

- **Primary Actor:** Player
- **Description:** The player starts the game.
- **Preconditions:** The player has launched the game via the binary.
- **Main Success Scenario:**
  1. The initial scene loads via `Alert` dialog with `PlayGame` button and `Exit` button.
  2. The player clicks the `PlayGame` button.
  3. The `Canvas` loads, containing a `Cannon`, a `Blocker`, and `9` `Target`s.
  4. The `shotCounter` is initialized to `0`.
  5. The `targetCounter` is initialized to `9`.
  6. The `Timer` starts counting down from `totalTime = 10` seconds.
- **Alternative Flows:**
  - Player clicks the `Exit` button, the game closes gracefully.
