## Use Case 001 - Play Game

- **Primary Actor:** Player
- **Description:** The player starts the game.
- **Preconditions:** The player has launched the game via the binary.
- **Main Success Scenario:**
  1. The player clicks the "Play Game" button.
  2. The scene loads the cannon, blocker, and blocks.
  3. The 10-second timer starts.
- **Alternative Flows:**
  - Player exits the game, game closes gracefully.
