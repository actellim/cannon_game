## Use Case 003 - Game Ends

- **Primary Actor:** Player
- **Description:** The game ends, score screen is displayed.
- **Preconditions:** The timer has run out or the player has destroyed all of the targets.
- **Main Success Scenario:**
  1. `Timer` reaches `0`.
  2. `Canvas` closes.
  3. `Alert` dialog displays loss, the `total` `ShotCounter`, the `remaining` `Target`s, and `totalTime`.
- **Alternative Flow:**
  1. `targetCounter` reaches `0`.
  2. `Canvas` closes.
  3. `Alert` dialog displays win, the `total` `ShotCounter`, and `totalTime`.
