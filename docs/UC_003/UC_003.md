## Use Case 003 - Game Ends

- **Primary Actor:** Player
- **Description:** The game ends, score screen is displayed.
- **Preconditions:** The timer has run out or the player has destroyed all of the targets.
- **Main Success Scenario:**
  1. Time runs out.
  2. Alert dialog displays loss, total number of shots fired, elapsed time.
- **Alternative Flow:**
  1. The player destroys all the targets.
  2. Alert dialog displays win, total number of shots fired, elapsed time.
